This branch contains all files and code related to the Home Page of the Basilico Restaurant Website project.

Purpose of This Branch
The homepage module focuses on:
Designing the main landing page for Basilico
Displaying restaurant highlights (specials, images, featured cuisine)
Creating a modern and responsive layout
Integrating homepage components with Order Online and Reservation pages

Folder Structure in This Branch
1. /homepage
Contains all code and styling specifically for the Home Page.

3. /admin
Included because some homepage content (like specials or promotional banners or testimonials) may be controlled or updated by the admin dashboard.

4. /order
Your homepage links to the Order Online module.
This folder is included so the navigation can be tested directly from the Home Page.

5. /reservation
Your homepage includes a Reserve a Table CTA button.
This folder allows seamless testing of the reservation flow.

5./assests
This folder stores all images used across the homepage and other linked modules.
