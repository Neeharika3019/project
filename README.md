Overview of the menu page

The menu page is interacted differently by admins and customers.

Files concerning admin:
1. admin_menu_items: This file lets the admin know he is in the admin control panel for the menu items. The admin can view the menu items available.
From this admin menu control panel, she can do different actions such as adding a new menu item, editing an existing menu item or even delete a menu item.

2. add_menu_item: This file lets the admin add a new menu item.
3. edit_menu_item: This file lets the admin edit an existing menu item.
4. delete_menu_item: This file lets the admin delete a menu item.
